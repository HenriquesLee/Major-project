Overview

This application demonstrates different learning paradigms across multiple domains:

  • Image Analysis
  • Video Analysis
  • Audio Analysis

Zero-Shot Learning

Zero-shot learning allows the model to make predictions for classes that were not seen during training.

How to use:

  1. Select "Zero-Shot Learning" mode
  2. Upload your file
  3. Click "Analyze"

One-Shot Learning

One-shot learning uses a single example to learn and make predictions about new instances.

How to use:

  1. Select "One-Shot Learning" mode
  2. Upload your query file
  3. Upload one example file
  4. Click "Analyze"

Few-Shot Learning

Few-shot learning uses a small number of examples to learn and make predictions.

How to use:

  1. Select "Few-Shot Learning" mode
  2. Upload your query file
  3. Upload multiple example files
  4. Click "Analyze"

API Reference

Endpoint: /api/predict/<domain>/<mode>

Parameters:

  • domain: image, video, or audio
  • mode: zero-shot, one-shot, or few-shot
  • file: The query file to analyze
  • example: (For one-shot) The example file
  • examples: (For few-shot) Multiple example files
